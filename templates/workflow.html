Workflow

Rapid Recon

Rapid Recon is a web based software that both dealerships use to track and communicate about pre-owned cars in inventory. Note that it is only for pre-owned and not new cars. This can be used in conjunction with the google sheets info to help you plan what cars need to be photographed, what cars to expect the next day, and to communicate to others that you've finished photographing a car. See the below modules for getting started with Rapid Recon.

vAuto

vAuto is an online software that you will use for uploading photos. Images uploaded here appear on the dealerships websites as well as various third party auto sales websites. It sometimes takes a full day for images to appear on websites outside of vAuto, so if a salesperson needs photos immediately, you will need to send them via email, dropbox, etc.
